#VakSiksha

VakSiksha is an AI-powered, voice-first educational platform designed to make learning accessible, interactive, and personalized for students across different backgrounds and languages. The platform leverages advanced Text-to-Speech (TTS) and conversational AI to transform traditional learning into a natural, dialogue-driven experience.

2)Clear explanation of API usage 
• We used text-to-speech FALCON murf api key for  the text-to-speech  version 
• We used dubbing murf api key for the audio dubbing in pro version 
• We used razor pay api key to access the pro subscription
• We used Murf AI Text-to-Speech API to generate voice output from text input.
• The application sends a POST request to the Murf API endpoint /v1/speech/generate with text, voice ID, and audio settings.
• The API returns an audio file which is then played in the web application.
